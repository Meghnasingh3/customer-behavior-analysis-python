# Customer Behavior Analysis (Python + BigQuery)

## Overview

This project analyzes customer purchasing behavior using large-scale e-commerce data. It combines SQL-based data extraction with Python analysis to generate business insights and customer segmentation.

---

## Objective

* Analyze customer spending behavior
* Segment customers based on purchasing patterns
* Identify high-value customer groups

---

## Tools & Technologies

* Google BigQuery
* Python (Pandas, Matplotlib, Seaborn)
* SQL

---

## Methodology

* Queried large datasets using BigQuery
* Calculated key metrics such as total spend and order frequency
* Performed customer segmentation into low, medium, and high-value groups
* Visualized results using Python

---

## Key Insights

* High-value customers contribute significantly to overall revenue
* Medium-value segment represents growth opportunity
* Low-value users require targeted engagement strategies
